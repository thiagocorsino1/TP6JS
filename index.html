1)¿Qué versiones de JavaScript hay en el mercado?.

Javascript naceria en 1995 con nombres provisionales como Mocha y Livescript. En 1997 se definiria el primer estándar de JavaScript a cargo de Ecma International que fue denominado ECMA-262(ES1) first edition también denominado JavaScript 1.2. Hasta la actualidad saldrian las siguientes versiones de ECMAscript:

  • ES2 en 1998
  • ES3 en 1999
  • ES5 en 2005
  • ES6 en 2015
  • ES7 en 2016
  • ES8 en 2017
  • ES9 en 2018
  • ES10 en 2019
  • ES11 en 2020
  • ES12 en 2021
  • ES13 en 2022

2)¿Cómo se integra JavaScript a HTML?.

Javascript se puede integrar de 3 maneras a html:

  • Con la etiqueta Script: se agrega la etiqueta script en el body y dentro de ella las sentencias.
  • En un Archivo Externo: se enlaza un archivo.js al html por un src.
  • En Linea: se agregan las sentencias dentro de una etiqueta html.

3)¿Cómo es el manejo de las variables, tipos de datos y expresiones regulares en JS?.

Variables

las variables se manejan con nombres simbólicos para darles valores en la aplicación. Los nombres de las variables, llamados identificadores, se ajustan a ciertas reglas. Un identificador de JavaScript debe comenzar con una letra, un guión bajo ( _ ) o un signo de dólar ( $ ). Las variables se pueden declarar usando var , const , o let.

Tipos de Datos

Se manejan los siguientes tipos de datos en JS

  • String:Cadenas de texto
  • Number:Valores numéricos.
  • Bigint:tipo de dato numerico que puede representar números enteros en el formato de precision arbitrario
  • Boolean:Representa una entidad lógica y puede tener dos valores: true y false.
  • null:Es un valor asignado tiene el valor de “no valor”.
  • undefined:Una variable a la que no se le ha asignado ningún valor tiene el valor undefined.
  • Symbol:Son siempre valores distintos aunque tengan el mismo nombre.
  • Object:Un valor en memoria al que podemos acceder por un identificador.

Expresiones Regulares

Las expresiones regulares se utilizan con los métodos de RegExp test y exec y con los metodos de String match , replace , search , and split.
